Как сделать шпору на стике в прошивке Bruce

1. Вход на GitHub

  • Зайдите на https://github.com
  • Авторизуйтесь под своим аккаунтом

2. Создание репозитория

  • Нажмите + → New repository
  • Заполните:
    • Repository name — любое
    • Description — по желанию
    • Visibility — Public
  • Включите Add README
  • Нажмите Create repository

3. Добавление файла

  • Измените README или добавьте новый файл
  • Обязательно используйте расширение .txt или .html

4. Сохранение

  • Впишите содержимое
  • Добавьте комментарий к коммиту
  • Нажмите Commit changes

5. Преобразование ссылки

Скопированная ссылка:

https://github.com/Fynex-x/test.git

Формат raw-ссылки:

https://raw.githubusercontent.com/ИМЯ_ПОЛЬЗОВАТЕЛЯ/ИМЯ_РЕПОЗИТОРИЯ/ИМЯ_ВЕТКИ/ПУТЬ_К_ФАЙЛУ

Пример:

https://raw.githubusercontent.com/Fynex-x/test/main/README.md

6. Использование в скрипте

Скачайте скрипт browser_bruse.js и откройте его для редактирования.

var website = [
    "https://raw.githubusercontent.com/Fynex-x/test/main/README.md",
    "https://raw.githubusercontent.com/Fynex-x/test/main/README.txt"
];

Примечание

Прошивка Bruce не поддерживает русские символы
